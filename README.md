# ecommerce-server

**Create user**
----

* **URL**

`/users/register` 

* **Method:**

`POST` 
  

*  **URL Params**

   **Required:**
 
`None` 

   **Optional:**
 
`None` 

* **Data Params**

`name, email, passwords` 

* **Success Response:**

  + **Code:** 201 <br />

    **Content:** `{ 
      "id": ...,
      "name": "...",
       "email": "...",
      "password": "...",
      "isAdmin": "...",
       "updatedAt": "...",
       "createdAt": "..."
     }`

 

* **Error Response:**

  + **Code:** 500 <br />

    **Content:** `{ msg : err.message }` 

  OR

  + **Code:** 400 <br />

    **Content:** `{ errors : [....] }`

*  **URL Params**

`None` 

   **Optional:**
 
`None` 

* **Data Params**

`email, passwords`

  + **Code:** 200 <br />

    **Content:** `{ 
      "name": "...",
      "token": "...",
    "isAdmin": Boolean
     }`

 

* **Error Response:**

    **Content:** `{ msg : err.message }` 

  OR

  + **Code:** 404 <br />

    **Content:** `{ "msg": "email / password is wrong" }`

OR

* **Code:** 401 <br />

    **Content:** `{ "msg": "invalid token" || "msg": 'JsonWebTokenError' }`
